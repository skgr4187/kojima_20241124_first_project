 ## アプリケーションURL
 打刻ページ：localhost/
 ログインページ：localhost/login
 会員登録ページ：localhost/register
 日付一覧：localhost/admin
  
  ## 能一覧
  ・会員登録
  ・ログイン
  ・ログアウト
  ・勤務開始
  ・勤務終了
  ・休憩開始
  ・勤務終了
  ・日付別勤怠情報取得
  （・ページネーション）
